## 프로젝트 설명
- Android에서 Compose를 사용하여 다양한 버튼과 이미지 상호작용을 보여주는 애플리케이션을 구현한 간단한 예제입니다.
- 각 이미지에는 클릭 시 카운터가 증가하는 버튼이 연동되어 있습니다. 이 버튼들은 사용자의 클릭에 반응하여 숫자를 업데이트하며, 이 과정은 `ViewModel`을 통해 상태 관리가 됩니다.
