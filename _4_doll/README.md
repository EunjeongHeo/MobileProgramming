## 프로젝트 설명
- 이 애플리케이션은 사용자가 여러 옷과 액세서리를 인형에게 입힐 수 있도록 하는 간단한 안드로이드 앱입니다.
- 사용자는 체크박스를 통해 각각의 아이템(팔, 귀, 눈썹 등)을 선택하거나 해제함으로써 인형의 모습을 변경할 수 있습니다. 앱은 화면 회전 시에도 사용자의 선택을 유지합니다.
